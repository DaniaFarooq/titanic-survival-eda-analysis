TITANIC SURVIVAL - EXPLORATORY DATA ANALYSIS

Project Overview:

This analysis explores the factors that influenced passenger survival on the Titanic.

Dataset Information:

  • Source: Kaggle Titanic dataset
  • Records: 887 passengers
  • Features: 8 original columns
  • Target Variable: Survived (0 = No, 1 = Yes)

Analysis Objectives:

  1. Perform comprehensive univariate analysis
  2. Identify key survival factors through multivariate analysis
  3. Create meaningful visualizations
  4. Derive actionable insights

print("\n6. Feature Engineering:")
print("=" * 40)

# Creating new features
df['Title'] = df['Name'].str.extract(' ([A-Za-z]+)\.', expand=False)
df['IsAlone'] = (df['FamilySize'] == 0).astype(int)
df['AgeGroup'] = pd.cut(df['Age'], 
                        bins=[0, 12, 18, 35, 60, 100], 
                        labels=['Child', 'Teen', 'Young Adult', 'Adult', 'Senior'])

# Survival rate by new features
print("\nSurvival rate by new features:")

# Travel alone vs with family
alone_survival = df.groupby('IsAlone')['Survived'].mean()
print(f"\nTraveling alone: {alone_survival[1]*100:.1f}% survival")
print(f"Traveling with family: {alone_survival[0]*100:.1f}% survival")

# Age group analysis
agegroup_survival = df.groupby('AgeGroup', observed=True)['Survived'].mean()
print(f"\nSurvival by age group:")
for agegroup, rate in agegroup_survival.items():
    print(f"   - {agegroup}: {rate*100:.1f}%")

KEY FINDINGS:

  2. Top Survival Factors (in order of impact):

    a. Gender: Female (68.1% survival) vs Male (31.9% survival)

    b. Passenger Class: 1st (63.0% survival) vs 3rd (24.2% survival)

    c. Fare: Higher fares correlated with better survival chances

    d. Family Size: Small families (2-4) had better survival than solo travelers
